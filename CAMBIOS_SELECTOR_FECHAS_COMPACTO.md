# Cambios: selector de fechas compacto

Se redujo el alto y ancho visual del selector de fechas usado en graficas y reportes para que no robe espacio a las graficas.

## Archivos modificados

- `frontend/src/styles/global.css`

## Ajustes incluidos

- Menor `padding` del contenedor.
- Inputs de fecha mas chicos.
- Botones `Actualizar` y `Restablecer` mas compactos.
- Texto de titulo/subtitulo reducido.
- Separacion menor entre campos.
- El selector sigue afectando solo la grafica, tabla o reporte donde aparece.
